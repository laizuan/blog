# 快速开始
它基于 ElementUI 2.x 版本上开发的组件库

### 依赖
- [Vue 2.x](https://cn.vuejs.org/v2/guide/)
- [ElementUI 2.x](https://element.eleme.cn/#/zh-CN/component/installation)
- [lodash 4.x](https://www.lodashjs.com/)

### 使用

### 全局设置

```js
Vue.use(UI, {
  request, // http 请求函数
  pageSize, // 表格每页显示个数选择器的选项设置
});
```
